# Pricing Exercise (NodeJS library)

# Runtime versions
The library was tested on Windows 10 with the following versions:

node version v10.8.0 

npm version 6.2.0 

# Installing instructions
### Clone the repo
```
git clone <repo-url>
```

### Install dependencies
```
cd <git-repo-parent-dir> 
npm install
```

# Running unit tests
```
cd <git-repo-parent-dir> 
npm test 
```
